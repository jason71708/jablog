> 此系列筆記為閱讀[官方文件](https://threejs.org/manual/#en/fundamentals)後的一些重點紀錄。

Three.js 是一個跨瀏覽器的指令碼，使用 JavaScript 函式庫或 API 來在網頁瀏覽器中建立和展示動畫的3D電腦圖形。Three.js 使用 WebGL。 --- 來自[維基](https://en.wikipedia.org/wiki/Three.js)
